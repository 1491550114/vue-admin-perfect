## 安装项目
```
# 克隆项目
git clone -b vue3.0-antdv https://github.com/chuzhixin/vue-admin-better.git
# 安装依赖
yarn install
# 本地开发 启动项目
yarn serve
```

### 打包
```
yarn build
```
